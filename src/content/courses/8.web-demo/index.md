A small demonstration course for the **web preview exercise** system:
lessons give you HTML, CSS, and JavaScript (or TypeScript) editor tabs with
a live page preview — build the page, watch it update as you type, and
submit your work when you're happy. Nothing is auto-graded; these exercises
are about *making* something.

*This course exists to exercise the feature during development; replace it
with real content (or fold these patterns into your own courses).*
