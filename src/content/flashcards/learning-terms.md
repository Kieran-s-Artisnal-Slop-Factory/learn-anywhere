---
title: Learning Science Terms
cards:
  - front: What is active recall?
    back: Deliberately retrieving information from memory instead of re-reading it — the retrieval attempt itself strengthens the memory.
  - front: What is spaced repetition?
    back: Reviewing material at increasing intervals, timed to just before you would forget it, instead of cramming.
  - front: Roughly how many chunks can working memory hold at once?
    back: About four.
  - front: What is interleaving?
    back: Mixing different topics or problem types within one study session rather than practicing one type in a long block.
  - front: What is a mnemonic?
    back: A memory device — acronym, rhyme, image, or story — that packages hard-to-remember information into a memorable form.
  - front: Familiarity vs. learning — what's the difference?
    back: Familiarity is recognizing material on the page; learning is being able to retrieve it with the book closed. Only the second predicts test performance.
